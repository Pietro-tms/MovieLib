# Biblioteca de Filmes

Aplicação web desenvolvida para consulta e exploração de informações sobre obras cinematográficas. A plataforma permite que os usuários naveguem por diferentes filmes e acessem detalhes fornecidos pela API do TMDB (The Movie Database).

O front-end foi construído com React.js, utilizando React Router para navegação entre páginas e Tailwind CSS para estilização da interface. A integração com a API do TMDB foi realizada por meio da Fetch API, nativa do JavaScript, permitindo a obtenção e exibição dinâmica dos dados dos filmes.

O projeto foi desenvolvido com foco em componentização, consumo de APIs REST, organização de código e criação de interfaces responsivas, proporcionando uma experiência intuitiva para os usuários.
